AMD Developer Hackathon ACT II / Track 1

Accuracy first. Remote tokens only when needed.

Track 1 Token Router is a CLI-first agent that uses fail-closed deterministic solvers, then validation-selected Kimi when authorized by the evaluator, with strict output validation and allowed-model fallback.

Why this wins

The hard part is not calling the strongest model. The hard part is knowing when not to call it.

75% binary

Evidence

A 571-task frozen ablation rejected the FunctionGemma/E2B, per-intent, matrix and Minimax challengers without post-test tuning.

571 tasks

Submission shape

Public GitHub repo, Docker, CLI reproduction, static demo and no-credit release path.

Demo command

ROUTER_MODE=fireworks \
python3 -m router ask "What is 6 * 7? Return only the number." --json

Expected route

solver_arithmetic, because code can answer this safely.

Expected answer

42, with final validation preserving the strict numeric format.
