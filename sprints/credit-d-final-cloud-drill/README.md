Ensaiar a submissao final no ambiente mais parecido possivel com o scoring oficial.

- [ ] Adaptar input/output oficial.

- [ ] Confirmar `stdout` limpo.
- [ ] Confirmar logs separados.
- [ ] Confirmar env vars finais.
- [ ] Congelar prompts.
- [ ] Congelar politica.
- [ ] Criar tag final.
- [ ] Atualizar README de submissao.

- O fluxo final roda do zero.

- A estrategia final esta documentada e defensavel.
