# Cover Image Brief

## Format

PNG or JPG, 16:9, readable at small size.

## Visual concept

A routing control room for AI inference: local AMD GPU path on the left, deterministic solver path below, Fireworks serverless audit on the right, and a central router choosing the cheapest accurate route.

## Text

Primary title:

`Track 1 Token Router`

Subtitle:

`Accuracy first. Remote tokens only when needed.`

## Style

- Technical, sharp and competition-ready.
- Avoid generic chatbot imagery.
- Use AMD-inspired red/orange accents only as subtle energy, not a fake logo.
- Show clean traces, arrows and token counters.

## Must not include

- Provider logos unless usage rights are clear.
- API keys, screenshots of dashboards or private IPs.
- Claims about final score before official evaluation.
